# Stochastic Calculus for Finance I – Python Implementation

This repository contains my implementations of core concepts from *Stochastic Calculus for Finance I* by Steven Shreve.

## What I’m doing
- Studying mathematical finance through binomial models
- Implementing pricing and hedging strategies in Python
- Building intuition behind arbitrage-free pricing

## Topics covered
- Binomial asset pricing model
- Risk-neutral probability
- Option pricing (European call/put)
- Replicating portfolio & delta hedging

## Goal
To develop a strong foundation in quantitative finance and prepare for quant/trading roles.
